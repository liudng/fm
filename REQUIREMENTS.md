# Linux 双面板文件管理器 - 需求规格

## 1. 简介

- **软件名称**：fm
- **适配系统**：仅 Linux
- **定位**：基于 Qt6 Widget 的双面板文件管理器，支持多选项卡、文件操作、自定义布局与配置持久化。

---

## 2. 用户界面布局

### 2.1 主窗口结构
- 顶部为菜单栏，下方为工具栏，剩余空间全部填充为双面板区域。
- 窗口缩放时双面板始终占满剩余空间。
- 窗口标题固定显示为 `"fm"`。
- 无地址栏、状态栏。

### 2.2 双面板
- 默认左右布局（左=面板1，右=面板2），可通过菜单切换为上下布局（上=面板1，下=面板2）。
- 面板间有分隔条可拖动调整大小，左右布局与上下布局的比例分别记忆。
- 隐藏一个面板后，剩余面板占满全部空间；重新显示时恢复原比例。
- 活动面板中的活动选项卡标题以粗体字指示。
- 点击面板任意区域（文件列表、选项卡栏等可获焦控件）将该面板切换为活动面板。

### 2.3 面板中的选项卡
- 每个面板可创建多个选项卡，用于显示不同目录。
- 默认左右面板各一个选项卡，均打开用户的 `Home` 目录。
- 选项卡标题显示当前文件夹名称：
  - 最多 16 个 UTF-8 字符，超过时显示前 15 个字符 + "…"。
  - 鼠标悬停显示完整路径 tooltip。
- 选项卡支持在同一面板内拖拽调整先后顺序。
- 选项卡区域提供：
  - "+" 新建按钮
  - "×" 关闭按钮（每个选项卡均有）
  - 选项卡过多时显示滚动按钮。

### 2.4 工具栏
- 关联当前活动面板的活动选项卡。
- 按钮功能与文件列表右键菜单项同步。
- 按钮启用/禁用状态与右键菜单项一致（随选中状态动态变化）。
- 工具栏按钮与右键菜单项共享快捷键。

## 3. 菜单功能

### 3.1 文件菜单
- **文件卷列表**：动态显示系统中所有已挂载的文件卷（不含未挂载块设备），每个卷一项，直接列在文件菜单下（非子菜单）。
  - 左键点击：在当前活动面板的活动选项卡中打开挂载点路径（不新建选项卡）。
  - 右键点击：弹出"安全卸载"和"弹出"两项。失败时弹错误对话框，文件菜单不关闭以便重试。
- **退出**：退出程序，保存当前布局。

### 3.2 收藏菜单
- **新建**：弹出 `QInputDialog` 输入框，输入名称后保存当前双面板的布局、各选项卡路径及排序依据到配置文件。
  - 名称重名时提示重新命名。
  - 收藏项支持右键删除（无需重命名）。
- **收藏项列表**：点击某一收藏项，先清空两个面板的所有选项卡，再恢复该收藏项保存的布局及路径。

### 3.3 设置菜单
设置菜单项均为全局配置，立即生效并保存到配置文件。提供两套入口：菜单快捷访问 + "设置"对话框综合管理（见 6.1）。

- **语言**：子菜单"英文"、"中文"，选中状态标识，全局切换界面语言。
- **主题**：子菜单列出 `QStyleFactory::keys()`（如 Fusion、Windows），选中状态标识，切换界面主题。
- **切换活动面板**：菜单项文字根据状态动态变化，在面板1与面板2间切换活动面板。
- **上下/左右结构切换**：菜单项文字根据当前状态切换布局。
- **显示/隐藏面板1** / **显示/隐藏面板2**：各自独立切换，菜单文字动态变化。
- **显示/隐藏.开头文件**：控制所有文件浏览选项卡（含新建）是否显示以 `.` 开头的文件/文件夹。
- **设置列**：进入设置对话框的文件浏览分组中的设置列（见 6.1）。
- **设置**：打开综合设置对话框（见 6.1）。

> 设置对话框打开时，设置菜单中的"立即设置"项**无需禁用**。

### 3.4 帮助菜单
- **关于**：弹窗显示软件名称、版本、作者、版权声明。

## 4. 面板与选项卡操作

### 4.1 选项卡上下文菜单
在选项卡标题上右键弹出：
- **关闭**：关闭当前选项卡（最后一个不可关闭）。
- **关闭其它**：关闭当前面板中除当前选项卡外的所有选项卡。
- **克隆**：在当前选项卡右侧新建一个相同路径的选项卡。

### 4.2 导航历史
- 每个选项卡独立维护导航历史。
- 记录进入子文件夹、点击 ".."、点击卷菜单项等操作。
- 后退/前进可用；后退后若执行新操作，则前进历史被截断。

## 5. 文件列表交互

### 5.1 显示与排序
- 以行列表展示，默认显示四列：**图标、文件全名、文件大小、最近修改日期时间**。
- 默认按文件名称排序（升序），点击列头切换排序方向（升序/降序），列头显示排序箭头。
- 排序规则：文件夹与文件混排，不区分类型；当某一列值相同时，自动使用上一次排序条件作为次要排序。
- 第一行固定为 `".."`：
  - 根目录 `/` 下隐藏。
  - 不参与排序。
  - 不可被选中（不参与多选）。
  - 在 `".."` 行上右键弹出"无选中"菜单。
  - Shift+范围选择时跳过 `".."`。
- 允许拖动列头分隔线调整列宽，允许拖动列头重排列顺序。
  - 列宽存储方式：
    - 所有列充满整行（无右侧空白）。
    - 除"文件全名"列以外的列使用**像素单位**存储列宽，最小宽度 20px。
    - "文件全名"列采用 Stretch 模式自动占据剩余宽度，不可直接拖拽调整，也不存储其宽度到配置文件。
  - 列宽、列顺序保存到配置文件（全局设置，调整一处影响所有选项卡）。
  - 新选项卡沿用保存的列宽与列顺序。

### 5.2 选择与操作
- **单击**选中项。
- **双击**：文件夹→在当前选项卡打开；文件→执行"打开"操作。
- 多选支持：
  - `Shift+点击` 范围选择（跳过 `".."`）。
  - `Ctrl+点击` 多选。
- 多选时仅适用于单选的项将被禁用，包括："打开"、"打开..."、"重命名"、"复制路径"、"复制文件名"、"属性"。

### 5.3 右键菜单
右键菜单根据是否有选中项动态变化：

#### 5.3.1 无选中项（或在空白区域右键，或在 ".." 行右键）
- 后退
- 前进
- 上一级
- 新建文件
- 刷新
- 新建文件夹
- 粘贴

#### 5.3.2 有选中项（在文件或文件夹上右键）
- **打开**：单个文件→系统默认程序打开；单个文件夹→当前选项卡内打开。
- **打开...**：单个文件→弹出选择应用程序打开；单个文件夹→新建选项卡（紧挨当前选项卡右侧）打开。
- **重命名**：仅单个文件/文件夹可用。
- **剪切**：剪切选中文件和文件夹（含子内容）到系统剪贴板。
- **剪切到对面**：剪切选中项到对面面板活动选项卡的文件夹（对面面板隐藏时禁用）。
- **复制**：复制选中文件和文件夹（含子内容）到系统剪贴板。
- **复制到对面**：复制选中项到对面面板活动选项卡的文件夹（对面面板隐藏时禁用）。
- **复制路径**：仅单个文件/文件夹可用，复制路径到系统剪贴板。
- **复制文件名**：仅单个文件/文件夹可用，复制文件名到系统剪贴板。
- **粘贴**：将剪贴板内容粘贴至当前目录。
- **移动到回收站**：将选中项移入回收站。
- **彻底删除**：弹出二次确认对话框后彻底删除。
- **属性**：仅单个文件/文件夹可用，弹窗显示详细信息（见 5.4）。

### 5.4 属性对话框
分组显示：
- **基本信息**：名称、路径、类型（`QMimeType::comment()`，如"目录"、"PNG 图像"）、MIME 类型、大小
- **用户与权限**：所有者、用户组、权限
- **时间戳**：创建时间、最近修改时间
- **inode 等系统信息**：inode、链接目标（符号链接仅显示直接目标，不做多层解析）等

### 5.5 新建文件/文件夹
- **新建文件**：默认名为"文件"，创建后自动进入重命名编辑状态。
- **新建文件夹**：默认名为"文件夹"，创建后自动进入重命名编辑状态。

### 5.6 打开方式（"打开..."）
- 通过 `xdg-mime` 列举可用程序（名称+图标），并提供"其他..."浏览自定义命令。
- 支持"记住此选择"复选框：
  - 记住对该 MIME 类型的选择，保存到配置文件 `[OpenWith]` section。
  - 下次进入此界面默认选中，但**不修改系统关联**。

### 5.7 键盘导航
文件列表支持标准键盘导航，所有操作纳入快捷键配置统一管理：
- `↑` / `↓`：选择上/下一行
- `Enter`：打开
- `Backspace`：上一级
- `Delete`：移到回收站
- `Shift+Delete`：彻底删除
- `F2`：重命名
- `F5`：刷新
- `Ctrl+C` / `Ctrl+V` / `Ctrl+X`：复制/粘贴/剪切
- `Ctrl+A`：全选

## 6. 设置与配置

### 6.1 设置对话框
左侧 1/5 宽度为设置分组 sidebar（界面、面板、文件浏览、快捷键），点选后右侧跟随显示对应分组内容。

底部按钮：
- **取消**：放弃自上次"应用"后未保存的更改并退出。
- **应用**：保存但不退出。
- **确定**：保存并退出。

#### 6.1.1 界面分组
- 设置语言：下拉框（英文、中文）
- 设置主题：下拉框（`QStyleFactory::keys()`）

#### 6.1.2 面板分组
- 布局切换：单选按钮组（左右 / 上下）
- 显示面板：复选框（面板1、面板2）
  - 至少需保持一个面板显示：禁用最后一个的取消操作。

#### 6.1.3 文件浏览分组
- 显示/隐藏.开头文件：复选框
- 设置列：带 checkbox 的列选择列表，可选列：
  - 图标
  - 文件全名
  - 文件大小
  - 类型（`QMimeType::comment()`）
  - MIME 类型（如 `inode/directory`、`image/png`）
  - 用户组
  - 所有者
  - 创建日期时间
  - 最近修改日期时间
  - 权限

#### 6.1.4 快捷键分组
- 表格列出所有功能及其快捷键，快捷键单元格可编辑（使用 `QKeySequenceEdit`）。
- 冲突项以红色字体显示。
- 涵盖范围：
  - 菜单栏所有菜单项（收藏菜单项及卷菜单项除外，因其为动态项）
  - 选项卡上下文菜单项（关闭/关闭其它/克隆）
  - 文件列表右键菜单项（所有变体）
  - 文件列表键盘导航（见 5.7）
  - Tab 焦点切换

### 6.2 配置文件
- **路径**：`~/.config/fm/config.ini`
- **格式**：INI

#### 6.2.1 Section 划分
| Section | 用途 |
|---|---|
| `[UI]` | 语言、主题 |
| `[Panels]` | 布局、显示面板状态 |
| `[File_Browser]` | 是否显示隐藏文件 |
| `[File_Browser_Columns]` | 各列可见性、列宽（像素）、列顺序 |
| `[Shortcuts]` | 快捷键映射 |
| `[Session]` | 面板布局/比例/选项卡/排序（退出时保存） |
| `[Favorites/<name>]` | 每个收藏项（名称特殊字符 percent-encoding） |
| `[OpenWith]` | "记住此选择"的 MIME→应用映射，格式：`<MIME类型>=<应用.desktop文件路径>` |

#### 6.2.2 存储说明
- 全局配置（`[UI]`、`[Panels]`、`[File_Browser]`、`[File_Browser_Columns]`、`[Shortcuts]`、`[OpenWith]`）独立存储，修改即生效。
- `[Session]` 在退出时保存，启动时恢复。
- 收藏项与 `[Session]` 使用相同数据格式，可互相转换。
- 收藏数量无上限。
- 收藏名含特殊字符时使用 percent-encoding（如 `/` → `%2F`，`=` → `%3D`）。

### 6.3 启动与配置错误处理
- 首次启动创建默认配置文件。
- 若配置文件损坏/解析失败：
  - **启动时**：提示读取失败，让用户选择是否重建。
    - 重建：使用默认配置，原文件备份（添加精确到毫秒的时间戳后缀，如 `config.ini.20260706_153045_123.bak`）。
    - 不重建：提示用户只能退出程序。
  - **运行中保存失败**：提示保存失败，让用户选择是否重建。
    - 重建：使用当前运行中配置，原文件备份。
    - 不重建：不退出程序，继续运行。
- 启动时检测快捷键冲突：
  - 仅第一个设置的快捷键生效。
  - 不给出错误提示（静默处理）。
  - 在设置对话框快捷键分组中以红色字体显示冲突项，供用户查看修改。

## 7. 其他交互细节

### 7.1 剪贴板
- 复制/剪切文件路径使用 URI 形式（`file:///path`），符合 X11 `text/uri-list` MIME 约定，与 Nautilus/Dolphin 行为一致，便于跨应用粘贴文件。
- 粘贴仅在剪贴板含有 `text/uri-list` 内容时启用。
- 剪切行为：未粘贴时再次剪切其他文件，旧剪切项恢复（取消剪切标记），新项成为剪切源。

### 7.2 文件操作与进度
- 删除/粘贴采用异步操作。
- 大文件操作（预估耗时 >1s）显示进度对话框，允许中途取消。
- 同名冲突（粘贴/剪切到对面）弹出选择：
  - 覆盖 / 跳过 / 重命名
  - 全部覆盖 / 全部跳过 / 全部重命名
- 权限不足或 IO 错误时弹出错误对话框。

### 7.3 热插拔与卷管理
- 菜单打开时实时枚举已挂载卷（通过 `QStorageInfo::mountedVolumes()`），每个挂载点独立显示为一项（含 btrfs 子卷的多个挂载点）。
- 卸载/弹出操作通过 `org.freedesktop.UDisks2` D-Bus 接口执行（设备文件路径自动转换为 D-Bus 对象路径）。
- 不显示未挂载块设备。

### 7.4 单实例运行
- 程序运行时为单实例。
- 新调用 `fm [path...]` 时：
  - 若已有实例运行，将 path 发送到已有实例的**活动面板**追加选项卡（通过 D-Bus 或本地 socket 通信）。
  - 若无实例运行，则正常启动。
- 命令行参数：`fm [path1] [path2]`
  - path1：追加到面板1的新选项卡
  - path2：追加到面板2的新选项卡
  - 仅 path1：只追加到面板1
  - 已有实例运行时：仅 path1 生效（追加到活动面板）

### 7.5 其他
- 不支持文件拖放（DnD）。
- 每个菜单项绑定快捷键（收藏菜单项及卷菜单项除外），快捷键配置存入配置文件，用户可修改。
- 工具栏按钮与右键菜单项共享快捷键。
- 关闭窗口（点 X）等同于"退出"菜单，保存布局并退出。
- 支持 Tab 键焦点切换（标准 Qt Tab 导航），纳入快捷键配置统一管理。

## 8. 技术细节

### 8.1 技术栈
- **框架**：Qt6 Widget，最低支持版本 Qt 6.8
- **构建系统**：CMake
- **开发环境**：Debian 13 (Trixie)
- **适配系统**：仅 Linux

### 8.2 图标
- 通过 `QIcon::fromTheme` 获取系统图标主题（如 Papirus、Adwaita）。

### 8.3 国际化
- 支持 i18n，内置中文与英文两种语言。
- 默认语言：英文。
- 语言/主题切换即时生效：
  - 语言：`QTranslator::load` + 重新翻译
  - 主题：`QApplication::setStyle` / `setPalette`

### 8.4 回收站
- 遵循 FreeDesktop.org Trash 规范（`~/.local/share/Trash`）。
- 支持外部分区的 `.Trash-1000`。

### 8.5 符号链接
- 不做多层解析，仅显示直接目标。

### 8.6 文件列表错误处理
- 若路径不存在或无权限，列表显示空 + 错误提示行。
- `".."` 行仍可点击返回上层文件夹。

### 8.7 列设置持久化
- 除"文件全名"列以外的列宽以像素单位保存；"文件全名"列自动占据剩余宽度。
- 列顺序保存。
- 新选项卡沿用保存的列宽与列顺序。

---

## 9. 待定/未实现项

- 文件拖放（DnD）：暂不支持，未来版本可能加入。
